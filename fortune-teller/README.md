# AI 算命大师 (Fortune Teller)

这是一个基于 React 和 Vite 构建的算命网页应用，集成了多种传统算命方法。

## 功能特点

*   **多维测算**：支持盲派八字、紫微斗数（模拟）、铁板神数（模拟）等多种算法。
*   **详细批注**：提供性格、事业、财运、婚姻、健康等多维度的运势分析。
*   **专业排盘**：使用 `lunar-javascript` 库进行精准的农历和八字排盘。
*   **精美界面**：古色古香的 UI 设计，沉浸式体验。

## 快速开始

1.  安装依赖：
    ```bash
    npm install
    ```

2.  启动开发服务器：
    ```bash
    npm run dev
    ```

3.  构建生产版本：
    ```bash
    npm run build
    ```

## 注意事项

*   本项目中的“紫微斗数”和“铁板神数”部分为演示性质的模拟算法，仅供娱乐。
*   “盲派八字”基于真实的干支历法排盘，但断语为通用模板。

## 技术栈

*   React
*   Vite
*   Tailwind CSS
*   Lunar-javascript
